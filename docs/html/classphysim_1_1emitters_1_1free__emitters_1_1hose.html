Detailed Description

A hose emitter.

This emitter tries to make the particles have the same behaviour as the water coming out of a hose. This so-called hose has been abstracted as a cone.

The cone is parametrised with a source, the vertex at the peak of the cone (see source), and with a circular base, centered at a point (see cc), that has radius r. The distance between the vertex and the center of the base is the height of the cone (see h). For easier calculations this object stores the perpendicular unit vectors v and w that span the plane containing the base.

In order to define this cone one has to use method set_hose_source. This method needs the source of the hose (the peak of the cone), a unit vector that points from the source to the base, the radius of the base r, and finally the height of the cone, h.

Particles are initialised with initial position source. The velocity is a random vector that goes from the source to a randomly generated point on the base of the cone. Therefore the minimum speed the particles will have is
