Detailed Description

Particle emitter classes.

This namespace contains a 'main' class, the emitters::base_emitter class that contains several, modifiable functions used to initialise the different attributes of particles.

This namespace also contains several other classes that implement the 'main' class. Each of these classes implement some of the 'main' class attributes to offer nice-looking particle systems.
