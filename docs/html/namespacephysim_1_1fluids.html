Detailed Description

All classes needed for fluid simulation.

This namespace contains the definition of what a fluid is, and the kernel functions for simulating a fluid's behaviour.

typedef std::function< float (float) > physim::fluids::kernel_scalar_function

Shorthand for a scalar kernel function.

A kernel function for fluids may be denoted as , where is the distance between two particles.

This function uses a maximum distance to define a neighbourhood for a given particle . Whenever this function is evaluated for a pair of particles such that the distance between them is larger than this maximum distance, it should return 0.

The only parameter is the squared distance between the particles.

Shorthand for a vectorial kernel function.

A kernel function for fluids may be denoted as , where is the distance between two particles.

This function uses a maximum distance to define a neighbourhood for a given particle . Whenever this function is evaluated for a pair of particles such that the distance between them is larger than this maximum distance, it should return 0.

  • Third parameter is the result of the function.
