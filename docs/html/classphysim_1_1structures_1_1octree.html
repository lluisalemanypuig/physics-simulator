Detailed Description

Octree class.

This is an implementation of a basic octree for both set of triangles (see function init(const std::vector<math::vec3>&, const std::vector<size_t>&, size_t) ),

and a set of vertices (see function init(const std::vector<math::vec3>&, size_t) ).

Builds the partition of a cloud of points.

For example, we may have a struct like this:

struct big {
   float a,b,c;
   char k;
   math::vec3 p1, p2, p3;
   char q;
   math::vec3 p4;
};

If one wanted to construct an octree using the math::vec3's p1, in the array

big b[512]

one would have to call this function as follows:

init(&b[0].b1.x, 512, sizeof(big))

Parameters
pPointer to the first element in which the math::vec3 is allocated.
nNumber of math::vec3.
offsetByte offset between math::vec3.
lodLevel Of Detail: minimum number of vertices per cell.

Builds a tree rooted at a node that partitions the triangles stored in triangles pointed by triangle_idxs.

Parameters
lodThreshold value for vertex partition. Below this amount, vertices will not be partitioned anymore.
vminPoint with the minimum value coordinates of the points in vertices.
vmaxPoint with the maximum value coordinates of the points in vertices.
verticesThe full list of non-repeated vertices of the triangles in triangles.
trianglesThe full list of triangles. Every three integer values we have one triangle.
tris_per_vertexThe i-th position of this vector contains the indices of the triangles adjacent to the i-th vertex. This is an auxiliary vector for faster tree building.
vertices_idxsThe list of vertices to be partitioned.
triangle_idxsThe list of triangles to incident to the node to be created. The indexes in this list are all multiples of 3 and point to positions (also multiple of 3) in triangles.
Returns
Returns the root of an octree that partitions the triangles in triangle_idxs and the points in vertices_idxs.
