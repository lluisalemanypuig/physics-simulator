Detailed Description

Class implementing a fluid.

A fluid is characterised by its volume, density and viscosity.

It is also characterised by a kernel function and its gradients (see kernel_density, kernel_pressure, kernel_viscosity).

Allocates memoery for n particles.

Allocates memory for n fluid particles and sets their initial mass.

The memory currently allocated is cleared only if n differs from the current number of paricles N.

It also initialises the mass of the particles and their index within the fluid.

◆ update_forces() [1/2]

virtual void physim::fluids::fluid::update_forces ( )
pure virtual

Update the forces generated within the fluid.

This method updates the forces acting on all particles. This update depends on the neighbouring information determined by the kernel functions.

This method does not update positions or velocities.

Precondition
The modification of a particles' force may assume that all particles start with null force (force equal to 0 in the three axes).
This method is called before computing the forces acting on the particles due to force fields.

Implemented in physim::fluids::newtonian.
