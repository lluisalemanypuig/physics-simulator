Detailed Description

Simulator class.

This class contains the algorihms used to simulate the movement of particles and systems of particles (like springs systems).

Use method apply_time_step to run the simulation step by step.

Use method add_geometry to add fixed geometrical objects (spheres, triangles, planes) with which the particles may collide with.

Function reset_simulation resets all particle to their original state using the initialiser objects (see free_global_emit, sized_global_emit).

Computes the forces acting in the simulation.

These forces are defined in the different focuses added to the simulator. Each force vector is added to the particle's force accumulator. The force accumulator, however, is not initialised to 0.

The result is set to the force acting on particle p.

void physim::simulator::simulate_free_particles ( )

Simulate free particles.

Calls _simulate_free_particles().

A free particle will only start 'living', i.e., moving in the environment, when its starting time (see free_particle::starttime) is equal or less than 0. Also, as long as free_particle::fixed is set to false. At every time step, the starting time of each particle is decreased by an amount equal to the time step (see dt).

The lifetime of each particle (see free_particle::lifetime) is also decreased by the exact same amount. Whenever this value reaches 0 the particle is reinitialised using the emitter for free particles (see free_global_emit).

Free particles collide with the geometrical objects added via method add_geometry(geometric::geometry *), and are moved depending on that geometry.

When activated, collisions of free particles against sized particles are checked.

void physim::simulator::simulate_sized_particles ( )

Simulate sized particles.

Calls _simulate_sized_particles.

The simulation of sized particles follows exactly the same rules as those for the simulation of free particles. The only difference is that collisions with geometrical object are copmuted considering that these particles have a certain size (see sized_particle::R).

When activated, collisions of free particles against sized particles are checked.

◆ free_global_emit

emitters::free_emitter* physim::simulator::free_global_emit
private

Initialiser applied to all free particles.

Whenever add_free_particle() or add_free_particles(size_t) are called, the attributes of the particles are initialised using this object.

This initialisation is also done whenever the lifetime of a particle goes below 0. Note that, in this case, some attributes like the velocity and position may not be zero, as a result of running the simulation.

The default behaviour of the function is to not do anything, that is, the attribtues of the particle are not modified at all.
