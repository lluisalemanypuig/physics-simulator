Detailed Description

Object partition class.

This is basically an implementation of an octree for a set of triangles, forming a connected triangular mesh or not.

Builds a tree rooted at a node that partitions the triangles stored in triangles.

Parameters
[in]verticesThe full list of non-repeated vertices of the triangles in triangles.
[in]trianglesThe full list of triangles. Every three integer values we have one triangle.
[in]tris_per_vertexThe i-th position of this vector contains the indices of the triangles adjacent to the i-th vertex.
[in]vertices_idxsThe list of vertices to be partitioned.
[in]triangle_idxsThe list of triangles to incident to the node to be created. The indexes in this list are all multiples of 3 and point to positions (also multiple of 3) in triangles.
