Detailed Description

Class that implements a plane.

A plane is defined with a normal, unit vector (see normal) and a constant term (see dconst).

A plane can be defined only with these two parameters (see plane(const math::vec3&, const math::vec3&) or with three points (see plane(const math::vec3&, const math::vec3&, const math::vec3&)).

Update a free particle in a collision with geometry.

Assumig that particle pred collided with this geometry, update its position, velocity, ... accordingly.

For example, some geometry may be 'bouncy', it may give the particle some extra speed. This needs to be updated in this method.

The particle pred is given in a state that is not modified by any solver. That is, the particle's position, velocity, force, ... have the value at time step T. The predicted position and velocity are given in pred_pos and pred_vel. These are predictions of the position and velocity for time T + dt (where dt is some time step). The modified particle due to the collision must be stored in pred.

This method is called only when it is sure that there has been a collision: the segment joining the particle's current position and the predicted position intersects the geometry.

Parameters
[in]pred_posThe predicted position of the particle.
[in]pred_velThe predicted velocity of the particle.
[out]predThe particle with the result of the collision.
